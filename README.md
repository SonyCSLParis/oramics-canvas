# Oramics Canvas

## Installation pour Cécile
Tu donnes les droits :

```bash
chmod +x run.sh
```

Une fois que c'est fait tu peux directement faire :
```bash
./run.sh
```

## Comment l'utiliser ?

### Waveform mode

1) Tu ouvres le VST dans Ableton
2) En gardant le VST ouvert tu dessines ta waveform
3) Tu joues avec la waveform

### Melody mode
1) Il faut que tu sois en play dans ta session Ableton (pour le moment)
2) Tu dessines ta modulation et t'appuies sur 'play'

NOTE: C'est un plugin qui send du midi donc il faut que tu ouvres une
midi track pour ce plugin spécifiquement et une autre track pour avoir
des effets midi
